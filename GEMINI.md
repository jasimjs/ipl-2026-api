# High-End Frontend Website Development

Welcome to the High-End Frontend Website Development project. This project focuses on delivering premium, high-performance, and SEO-optimized web experiences using modern frontend technologies and AI-assisted design workflows.

## Project Mandates

1.  **Aesthetics & UX:** Prioritize visually stunning, modern, and interactive user interfaces. Every component should feel "alive" with consistent spacing, typography, and interactive feedback.
2.  **Design-First Workflow:** Always follow a structured design process: Design System -> Low-Fidelity -> High-Fidelity -> Implementation.
3.  **SEO Standards:** Every page must adhere to high-quality SEO standards, including semantic HTML, meta tags, schema markup, and optimized assets.
4.  **Performance:** Optimize for Web Vitals (LCP, FID, CLS). Use efficient bundling, image optimization, and lazy loading.
5.  **Accessibility:** Ensure all components are WCAG compliant.
6.  **Technical Excellence:** Adhere to the standards defined in [.gemini/conventions/index.md](.gemini/conventions/index.md).

## Specialized Workflows

### 1. Design & Discovery
- **Reference Gathering:** Always ask the user for a reference design (URL, image, or description). If none is provided, proceed with an original high-end design.
- **Prompt Templates:** Use the specialized templates in [.gemini/prompts/index.md](.gemini/prompts/index.md) for consistent results with Stitch MCP.
- **Design System:** Use the `stitch` MCP to define a comprehensive design system (colors, typography, spacing, components).
- **Low-Fidelity Design:** Create a low-fidelity wireframe using `stitch` MCP and seek user confirmation before proceeding.
- **High-Fidelity Design:** Once confirmed, create a high-fidelity design using `stitch` MCP, focusing on premium aesthetics.

### 2. Implementation
- **Component-Driven:** Build modular, reusable components.
- **Vanilla CSS / Modern CSS:** Prefer Vanilla CSS or modern CSS-in-JS solutions that allow for maximum design flexibility, avoiding restrictive utility-first frameworks unless requested.
- **SEO Optimization:** Implement SEO best practices during the development of each page.
